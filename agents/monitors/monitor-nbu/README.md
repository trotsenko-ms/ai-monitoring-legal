# monitor-nbu

**Версія:** 1.0
**Модель:** claude-sonnet-4-5
**Тип:** Monitor
**Pipeline:** Гілка держорганів (запускається orchestrator-state)

## Призначення

Моніторинг опублікованих нормативних актів Національного банку України у сфері ЕДО, електронних підписів, архівного зберігання та банківського документообігу.

## Джерело

`https://bank.gov.ua/ua/legislation`

## Тематика

- Електронний документообіг (ЕДО) у банківській сфері
- КЕП / УЕП / електронні довірчі послуги
- Архівне зберігання документів (строки зберігання, ЕПК НБУ)
- Банківська документація та звітність
- Управління документами / номенклатура справ

## Ключові відстежувані акти

- Постанова НБУ №130 — перелік документів / строки зберігання
- Постанова НБУ №73 — галузевий державний архів НБУ
- Постанова НБУ №85 — ЕПК архіву НБУ
- Постанова НБУ №303-рш — документування управлінської інформації

## Розклад

Запускається orchestrator-state двічі на день: 08:00 і 17:00 (Київ). Підтримує manual trigger «НПА».

## Логіка роботи

1. Визначає розрахунковий період (від попереднього запуску до поточного)
2. Завантажує seen_urls для дедуплікації
3. Отримує список актів НБУ з bank.gov.ua/ua/legislation за розрахунковий період
4. Фільтрує дублікати та перевіряє релевантність тематиці
5. Зберігає JSON-артефакт і оновлює seen_urls
