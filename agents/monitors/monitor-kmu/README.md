# monitor-kmu

**Версія:** 1.0
**Модель:** claude-sonnet-4-5
**Тип:** Monitor
**Pipeline:** Гілка держорганів (запускається orchestrator-state)

## Призначення

Моніторинг опублікованих постанов та розпоряджень Кабінету Міністрів України у сфері ЕДО, електронних підписів, архівного зберігання, банківського документообігу та цифровізації державних послуг.

## Джерело

`https://www.kmu.gov.ua/npasearch`

## Тематика

- Електронний документообіг (ЕДО)
- КЕП / УЕП / електронні довірчі послуги
- Архівне зберігання документів
- Банківський документообіг
- Цифровізація / міжвідомча електронна взаємодія (Трембіта, СЕВДЕІР)

## Розклад

Запускається orchestrator-state двічі на день: 08:00 і 17:00 (Київ). Підтримує manual trigger «НПА».

## Логіка роботи

1. Визначає розрахунковий період (від попереднього запуску до поточного)
2. Завантажує seen_urls для дедуплікації
3. Отримує список актів КМУ з kmu.gov.ua/npasearch за розрахунковий період
4. Фільтрує дублікати та перевіряє релевантність тематиці
5. Зберігає JSON-артефакт і оновлює seen_urls
