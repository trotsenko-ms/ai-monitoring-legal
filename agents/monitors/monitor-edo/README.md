# monitor-edo

**Версія:** 1.0
**Модель:** claude-sonnet-4-5
**Статус:** active (мігровано з v1.0)

## Призначення

Моніторинг змін нормативно-правових актів України у сфері:
- електронного документообігу (ЕДО)
- електронних підписів (КЕП, УЕП, ЕП)
- архівного зберігання документів
- банківського документообігу

## Джерела пошуку

- `zakon.rada.gov.ua`
- `kmu.gov.ua`
- `bank.gov.ua`

## Дедуплікація

Файл: `data/dedup/seen_urls_edo.json`
Час зберігання записів: 90 днів

## Розклад

| Тип | Час | Тригер |
|-----|-----|--------|
| Cron | 08:30 (Київ / UTC+3) | — |
| Manual | за запитом | команда «ЕДО» |

## Pipeline

```
monitor-edo → validator-edo → email [ЕДО]
```
